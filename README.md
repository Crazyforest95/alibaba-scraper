# Scrape Alibaba.com search results using Scrapy

## Requirements 
- Python 3 
- Scrapy

## Running the Scraper

1. Add search keyword to keywords.csv 
2. Run command `scrapy crawl alibaba_crawler -o out.csv -t csv` to get data as CSV into a file called out.csv
